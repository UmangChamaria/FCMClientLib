initialise

fun initialise(application: Application, logLevel: Logger.LogLevel = Logger.LogLevel.ERROR, retryInterval: Long = 30): Unit

Initialize the FCM client. This needs to be called from the onCreate() of Application class.

Parameters

application - instance of the Application

logLevel - optional parameter takes in the level of logs which should be printed by the SDK when the application is running in debug mode. By default, only error logs are printed. Refer to Logger.LogLevel for more details.

retryInterval - optional parameter which takes the time interval(in seconds) after which the SDK should retry registering for Push Token in case of any failure. Default value is 30 seconds.
